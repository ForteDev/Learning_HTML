HTML 속성 공부

이 문서는 다음 링크를 공부하면서 끄적거린 HTML파일입니다.

HTML 속성

  • 모든 html 태그는 속성(attribute)를 가질 수 있습니다.
  • 속성은 html 요소에 부가적인 정보를 더할 수 있습니다.
  • 속성은 항상 시작 태그에 명시됩니다.
  • 속성은 대개 이름 = "값"처럼 이름/값의 켤레 형태로 나타냅니다.

이 링크에서 알려주는 HTML 속성은 다음과 같습니다.

  • href
  • src
  • width & height
  • alt
  • style
  • lang
  • title

이제 하나하나 연습해보겠습니다. (말이 엄청 딱딱하구먼)

이것은 href 속성이오

href 속성은 a 태그에 쓰는 속성이오. <a href="링크"> 의 형태로 속성을 사용하오. 따옴표로 안 묶어도 인식되오. 하지만 그 딴식으로 게으름 피우면 큰 코 다치오. 무조건 따옴표로 묶도록 하시오. 또한 작은 따옴표를 써도 좋고 큰 따옴표를 써도 좋으니 독자들 편한 방식으로 사용하시오. 아 대부분 큰 따옴표를 사용하긴 하오. 하, 참 이렇게 쓰는 경우도 있소. href="큰 따옴표가 '작은 따옴표'를 둘러쌌소". 소인이 실수를 범했소. 작은따옴표와 큰따옴표를 같이 쓰는 경우를 여기서 설명해서는 안되었소. 송구하오.

이것은 src 속성일세

자네, src 속성을 배우러 왔구먼. 아주 자~알 왔네. 자 나와 함께 src 속성을 배워봅세. src 는 영어로 '쏘:스' (source)라는 말의 로마자식 준말이네. 이 '쏘:스'라는 말은 국문으로 출처를 의미하네. 엣티엠엘(html)을 만든 양놈이 '이건 사진의 출처이노라.' 하는 표시로 이런 이름을 붙인 것 같네. 허허 여하튼! 설명을 계속하겠네. src는 <img> 태그와 함께 쓰인다네. <img src="사진의 링크를 여기에 입력하게"> 이렇게 쓰이네. 여기에 링크를 사용하는 방식이 여러 가지가 있고 다소 복잡하기도 하네. 설명하다간 내가 공부할 시간이 없으니 생략하도록 하겠네. 이해합세.

짐이 고하노라

짐이 고하노라. 해당 방식으로 엣티엠엘을 설명하기에는 시간이 너무 부족하니라. 이 깊은 뜻에 의거해 숭고한 결정을 내렸노라. 나머지 속성은 설명을 생략하겠노라. 머리속에 집어넣는 학습만 하면 되니 속히 공부할 수 있을 것이니라. 신속히 공부하고 서술은 추후에 하도록 하라.
