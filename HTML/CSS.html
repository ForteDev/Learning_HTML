CSS란?

CSS는 Cascading Style Sheets를 의미합니다. CSS의 탄생은 HTML로만 작업했던 웹의 초창기에 변혁을 일으켰다고 합니다. 수많은 작업을 줄여줘서 더 효율적으로 웹페이지를 작성하게끔 만들었습니다. 그리고 HTML이 웹페이지를 구조화하는 본연의 역할만을 수행하고 꾸미고 단장하는 역할을 CSS가 맡게되었습니다. 웹페이지를 최적화하여 더 적은 양의 데이터만으로 웹페이지를 로딩하게 만들었으며, 다수의 웹페이지의 양식과 구성을 단숨에 변경하게 도와줍니다.
CSS를 본격적으로 배우기 전이지만 CSS가 어떻게 동작하는지 약간의 감이 오내요. .CSS 파일을 따로 만들어 CSS 파일에 웹페이지 레이아웃, 디자인들을 모두 입력하는 것 같습니다. 그리고 모든 HTML 문서가 .CSS 파일을 참조하도록 만든다면 모든 웹페이지가 동일한 레이아웃과 디자인으로 나타나는 게 아닐까요? 웹 페이지 디자인을 바꾸고 싶다면 HTML 문서를 하나하나 수정할 필요없이, .CSS 파일만 수정한다면 해당 파일을 참조하는 모든 HTML 페이지에 똑같이 반영되겠죠. 간단하게 추리할 수 있는 당연한 말을 적어보았습니다.

CSS로 HTML 꾸미기

CSS는 HTML 요소가 화면과 종이 등 여러 매체에 표시되는 방식을 기술합니다. CSS를 세가지 방식으로 나타낼 수 있습니다.

  • 코드 내에서 - HTML 요소에 style 속성을 사용합니다.
  • HTML 안에서 - HTML head 태그 안에 style 요소를 사용합니다.
  • HTML 밖에서 - 외부 CSS 파일을 사용합니다.
CSS를 추가하는 가장 일반적인 방식은 외부 CSS파일을 작성하는 것입니다. 하지만 실습을 위해서 코드 내에서 그리고 HTML 안에서 CSS를 사용하겠습니다.

Inline CSS

특정 HTML 요소에만 CSS를 적용할때 Inline CSS를 사용해요. style 속성을 사용해서 표현한답니다.

Internal CSS

HTML 웹페이지 하나에만 CSS를 적용할 때 Inline CSS를 사용해요. head 태그 부분에 style 태그 요소를 표현합니다. style 태그로 둘러싸고, 지금 보고 있는 화면도 내부 CSS를 통해 나타낸 것입니다.
