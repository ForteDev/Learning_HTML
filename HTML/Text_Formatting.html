HTML Text Formatting - HTML 택스트 구현

이 문서는 https://www.w3schools.com/html/html_formatting.asp에 근거해있습니다.



  • <b> - 볼드체 표시
  • <strong> - 중요성 표시
  • <i> - 이텔릭체 표시
  • <em> - 강조 표시
  • <mark> - 마크(형광펜) 표시
  • <small> - 작게 표시
  • <del> - 취소선 표시
  • <int> - 삽입된 글자 표시
  • <sub> - 줄 아래에 글자 표시
  • <sup> - 줄 위에 글자 표시


  웹 브라우저는 <strong>을 <b>로, <em>을 <i>로 표시합니다. 두 태그에 차이점이 있는데요. <b>와 <i>는 각각 볼드체와 이텔릭체를 정의하지만, <strong>과 <em> 태그는 해당 텍스트가 중요하다는 의미를 정의합니다.
