HTML 표 정의하기

HTML 표는 <table> 태그를 사용해 정의합니다.

표의 행(가로줄)은 <tr>태그로 정의하고, 표의 머리글 행(말로 설명하기 쪼금 어려운데, 표의 첫번째 행에 '나이', '성별', '직업' 과 같이 아래에 적힐 내용이 뭔지 적는 부분이라고 할까나)은 <th>태그로 정의한다. 그리고 표의 내용과 칸은 <td> 태그로 정의한다.

기본 설정으로 <th> 요소는 굵은 글씨체에 가운데 정렬이 된다.

기본 설정으로 <td> 요소는 보통 글씨체에 왼쪽 정렬이 되어있다.
