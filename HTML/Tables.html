HTML Tables - HTML 표

오랜만에 작성하는 HTML 문서입니다. https://www.w3schools.com/html/html_tables.asp의 내용을 토대로 웹페이지를 작성했습니다.

HTML 표 정의하기

HTML 표는 <table> 태그를 사용해 정의합니다.

표의 행(가로줄)은 <tr>태그로 정의하고, 표의 머리글 행(말로 설명하기 쪼금 어려운데, 표의 첫번째 행에 '나이', '성별', '직업' 과 같이 아래에 적힐 내용이 뭔지 적는 부분이라고 할까나)은 <th>태그로 정의한다. 그리고 표의 내용과 칸은 <td> 태그로 정의한다.

기본 설정으로 <th> 요소는 굵은 글씨체에 가운데 정렬이 된다.

기본 설정으로 <td> 요소는 보통 글씨체에 왼쪽 정렬이 되어있다.

주의: <td> 요소는 표에 들어있는 자료의 그릇입니다.
<td> 요소는 글, 그림, 목록, 또 다른 표, 기타 등등 모든 종류의 HTML 요소를 담을 수 있습니다.

HTML 표 - 테두리 추가하기

표에 테두리를 추가하려면, CSS border 특성을 사용해야 합니다

border: 1px solid black; 특성을 사용하겠습니다.

HTML 표 - 테두리 겹치기

border 특성만 사용하면 상단의 표처럼 테두리가 중구난방으로 나눠져 깔끔하지 않다. 그래서 CSS border-cllapse 특성을 사용합니다.
border: 1px solid black;
border-collapse: collapse 로 설정해보겠습니다.
