Testing CSS - CSS 검토하기

외부 CSS 공부하기

CSS 파일은 여러 HTML 페이지를 한꺼번에 정의하기 위해 사용합니다. 하나의 외부 CSS 파일로 모든 웹사이트를 한번에 변경할 수 있습니다! 외부 CSS를 HTML 파일에 적용하려면 head 태그내에서 <link rel="stylesheet" href="CSS파일명.css">태그를 삽입해야합니다. link 태그를 타고 외부 CSS 파일이 소환되고 외부 CSS 파일에 적힌 값대로 웹페이지가 꾸며집니다.
CSS에서 글꼴을 설정하려면 font-family 특성을 사용하고, 글자 크기를 조정하려면 font-size 특성을 사용하면 됩니다. 한글 글꼴을 나타낼 수도 있습니다.

CSS Padding(칸 여백)을 사용해서 border로 구분한 칸 속 글자와 칸 사이의 간격을 설정할 수 있습니다. CSS Margin (페이지 여백)을 사용해서 해당 HTML 요소(태그)와 화면 테두리 사이의 간격을 설정할 수 있습니다. 두 친구 모두 픽셀(px)값으로 크기를 조정합니다.
그리고 같은 요소여도 id 값으로 구분해서 다르게 표현할 수 있습니다. 지금 보고 계시는 문단은 #p01 이라는 스타일을 따로 정의해서 꾸몄습니다. 요소 id는 페이지 안에서 고유해야 합니다. 고유 요소 하나만 꾸며주기 위해 사용해야합니다.

The class Attribute - 클래스 속성

CSS에 클래스를 적용하면, 각 요소의 성격과 특성에 맞게 구분해서 꾸며줄 수 있습니다. 예를 들어, 웹페이지를 서론 본론 결론으로 나눠서 구별해서 꾸며준다고 생각해볼까요? 이때 서론 문단 맵시에 해당하는 클래스 본론 문단 맵시에 해당하는 클래스 결론도 마찬가지로 구별된 클래스로 나눠서 정의한다면, 각 문단마다 알맞는 디자인으로 표현할 수 있을겁니다. 그럴때 <p class="클래스 이름">과 같이 표현해줘요. 이렇게 class 속성으로 표현하면 따로따로 표시할 수 있습니다. css 파일에서는

p.클래스이름 { 스타일 특성 } 과 같은 형태로 정의해줘요.
